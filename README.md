# Agrivar Sustainability Reporting Portal

## Overview
This project is a web-based sustainability reporting portal developed as part of an academic bachelor’s thesis in *Web Technologies for Business Sustainability*. The application provides a digital interface for the transparent publication and download of sustainability reports of a company operating in the primary sector, with a specific focus on accessibility, clarity of information, and stakeholder communication.

The case study presented in this project refers to **Agrivar**, an agricultural company based in the province of Bologna, active in olive groves, vineyards, and typical local crops, as well as agritourism activities. The portal aims to support corporate transparency by making sustainability data and documents easily accessible through a modern web interface.

## Objectives
The main objectives of the project are:
- To design and develop a responsive web page for sustainability reporting.
- To improve transparency in the primary sector through digital tools.
- To enable the structured publication and download of sustainability documents.
- To present key sustainability indicators through visual and interactive elements.

## Features
- Responsive single-page layout based on modern web standards.
- Dedicated section for downloading sustainability reports and related documents.
- Data visualization through interactive charts.
- Clear separation of informational, analytical, and documentary content.
- User-friendly navigation designed for different stakeholder profiles.

## Technologies Used
- **HTML5** for semantic structure and content organization.
- **CSS3** for styling and responsive layout.
- **Bootstrap 5** for grid system, components, and mobile-first design.
- **JavaScript (ES6)** for interactive behavior.
- **Chart.js** for data visualization and graphical representation of sustainability indicators.

## Usage
The project is a static web application and does not require a backend server.

To run the project locally:
1. Clone the repository.
2. Open `index.html` in a modern web browser.

All functionalities, including navigation, charts, and document downloads, are available directly in the browser.
